# B10773_영화 감독 숌
666은 종말의 수이다. N을 입력받아 666이 N번째 나오는 수를 출력하라

- 가장 중요했던 코드  
<pre>
if (num % 1000 == 666)
{
	i++;
	continue;
}

for (int j = 0; j < 5; j++)
{
	num = num / 10;
	if (num % 1000 == 666)
	{
		i++;
		break;
	}
} 
</pre>
마지막이 666이면 1000을 나눠 나머지를 구할 수 있고,  
마지막이 666이 아니면 10을 나눠서 다시 진행하여 종말의 수가 있는 지 체크한다.
